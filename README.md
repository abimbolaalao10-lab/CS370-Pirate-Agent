# CS 370 – Pirate Intelligent Agent (Reinforcement Learning)

## Project Overview
In this project, I developed a pirate intelligent agent using deep Q-learning to solve a pathfinding problem in a treasure hunt game. The goal of the agent was to navigate a maze environment and reach the treasure efficiently while avoiding obstacles.

The project demonstrates the application of reinforcement learning and neural networks to a real-world-style problem.

---

## Work Completed

The starter code provided included:
- The maze environment (TreasureMaze)
- The experience replay system (GameExperience)
- Basic structure for training the agent

I implemented:
- The deep Q-learning training algorithm (qtrain function)
- The decision-making logic using an epsilon-greedy strategy
- Training loops for learning optimal paths
- Integration of neural networks for Q-value prediction
